---
title: "Unable to connect when using API extraction parameters for Unlimited Proxy?"
---

- The IP whitelist has not been added.
- The whitelisted IP is from mainland China (rejected).
- The actual outbound IP does not match the whitelist.
- On your computer's cmd, run `curl myip.ipip.net` or `curl ipinfo.io` to confirm your real public IP and geographic location, then compare with the whitelist settings.
