---
title: "Can YumiProxy be connected directly?"
---

Hello. Before using our proxy IPs, please ensure that you are in an overseas network environment, because mainland China networks do not support direct connections.

- If you are currently using a mainland network, please first connect to an overseas environment through a front‑end network, and then use our proxy IPs.
- If you are already using an overseas network, you can connect to our proxy IPs directly.
